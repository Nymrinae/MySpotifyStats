## Stack

- Nuxt.js
- Vue.js
- VueX (state management)

## Features
- Check Top Artists (4 weeks, 6 months, all time)
- Check Top Tracks (4 weeks, 6 months, all time)
- Check playlists and songs on playlists
- Check the history of your songs
